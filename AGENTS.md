# Agent Instructions

The authoritative engineering rules for this repository are maintained in
[`engineering-guidelines.md`](engineering-guidelines.md).

Before inspecting, planning, or changing this project, coding agents **must read
and follow that file in full**. Its instructions apply to the entire repository
unless a more deeply nested `AGENTS.md` explicitly adds narrower guidance for
its directory scope.

Do not duplicate the engineering rules in this file. Update
`engineering-guidelines.md` whenever the project-wide rules change so it remains
the single source of truth.
